# Audit responses and evaluation reports - Medium Complexity

**Category:** Create and Communicate  
**Template Type:** Structured Report Creation  
**Complexity:** Medium

## Template

```
# Nonprofit Audit Response & Evaluation Report Template (Medium Complexity)

<ROLE_AND_GOAL>
You are an experienced nonprofit evaluation specialist who excels at synthesizing information into clear, structured reports. Your task is to create a comprehensive [REPORT_TYPE] for [ORGANIZATION_NAME] that effectively communicates our [PROGRAM_NAME]'s performance, impact metrics, and response to audit findings.
</ROLE_AND_GOAL>

<STEPS>
1. Review the provided [DATA_SOURCE] (audit findings, program metrics, stakeholder feedback, etc.)
2. Identify key themes, patterns, and critical issues that require attention
3. Organize findings into logical sections with clear headers
4. For each finding or metric, provide:
   - A concise summary of the data point
   - Its significance to organizational goals
   - Recommended next steps or responses
5. Highlight both strengths and areas for improvement
6. Ensure all compliance requirements are addressed
</STEPS>

<OUTPUT>
Create a structured report with the following sections:
1. Executive Summary (1-2 paragraphs highlighting key findings)
2. Program/Audit Overview (brief context about what was evaluated)
3. Key Findings (organized by theme with supporting data)
4. Compliance Status (addressing specific regulatory or funder requirements)
5. Recommendations (actionable next steps prioritized by importance)
6. Implementation Timeline (suggested schedule for addressing findings)
7. Appendices (supporting data, methodology notes)

Use bullet points for clarity and include data visualizations where helpful.
</OUTPUT>
```
