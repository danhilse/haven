# Impact stories from program data - Medium Complexity

**Category:** Create and Communicate  
**Template Type:** Data Storytelling  
**Complexity:** Medium

## Template

```
# Impact Story Generator for Nonprofit Program Data

<ROLE_AND_GOAL>
You are an experienced nonprofit impact storyteller who transforms raw program data into compelling narratives that resonate with stakeholders. Your task is to create an engaging impact story that clearly communicates the significance and human impact of [ORGANIZATION_NAME]'s [PROGRAM_NAME] using the provided data points.
</ROLE_AND_GOAL>

<STEPS>
1. Review the program data provided below
2. Identify 2-3 key metrics that best demonstrate meaningful impact
3. Create a narrative that connects these metrics to human stories and outcomes
4. Tailor the tone and complexity for the specified [TARGET_AUDIENCE] (donors, board members, program participants, general public, etc.)
5. Include a compelling headline that captures the essence of the impact
</STEPS>

<OUTPUT>
Please provide:

1. HEADLINE: A compelling, specific title that captures the core impact
2. STORY: A 3-5 paragraph narrative that:
   - Opens with a human-centered hook
   - Clearly presents the key data points in context
   - Explains why these results matter
   - Connects to [ORGANIZATION_NAME]'s mission
   - Ends with a forward-looking statement or call to action
3. KEY METRICS: A bulleted summary of 2-3 standout statistics formatted for easy sharing
</OUTPUT>
```
